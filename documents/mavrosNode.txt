以下记录为有关mavros节点的相关信息
部分参考资料来源
   https://mavlink.io/en/messages/common.html
   rosmsg是mavros_msgs的，具体信息可以从这里找
   （http://wiki.ros.org/mavros 这个也有一些）

   rosmsg是sensor_msg或者其他的，可以直接把消息类型百度找到docs.ros.org提供的具体解释（这个对应的代码应该在/opt/ros/kinetic/的目录下）

  fcu：飞行控制组件 flight control unit

ROS的通信方式主要包括两种：话题（topic）和服务（service），我觉得这两种通信方式的最主要的不同在于同步和异步的区别
  topic
    可以看做是一种异步的通信方式，由master节点掌管，发布消息的节点将对应的消息使用advertise发布，需要该消息的节点在需要的时候调用subscribe查询master节点
    topic支持一对一，多对多的通信方式
    topic处理连续的数据流
    callback函数在服务端启动，因为是异步的，当服务端什么时候收到信息什么时候就做事
    topic中的数据只要准备好了，就可以被callback——回调函数调用。而发送端的topic的发送时机由publisher自己决定
  service
    是一种同步的通信方式，使用call方法调用，响应它直到数据被对方接受到。有时候这也可以用来发送指令（传回的消息可以代表动作成功还是失败）
    用于那些快速停止的程序，如查询一个node 的状态，或是做数学运算，。它不应当被用于那些需要长时间运行的进程中，特别是有异常处理的子进程。
    绝不应该用于状态的变化。

 * /mavros/home_position/home [mavros_msgs/HomePosition]
   This message can be requested by sending the MAV_CMD_GET_HOME_POSITION command. The position the system will return to and land on. The position is set automatically by the system during the takeoff in case it was not explicitly set by the operator before or after. The position the system will return to and land on. The global and local positions encode the position in the respective coordinate frames, while the q parameter encodes the orientation of the surface. Under normal conditions it describes the heading and terrain slope, which can be used by the aircraft to adjust the approach. The approach 3D vector describes the point to which the system should fly in normal flight mode and then perform a landing sequence along the vector.

 * /mavros/hil/optical_flow [mavros_msgs/OpticalFlowRad]
  Optical flow from a flow sensor 
  optical flow：光流
  光流（Optical Flow）是一种研究图像对齐的算法，一般包括两大类：稀疏光流和稠密光流。顾名思义，稀疏光流就是研究图像中稀疏点的光流，这些点一般是角点；稠密光流则是研究图像中所有点的偏移量。

 * /mavros/hil/rc_inputs [mavros_msgs/RCIn]
  Sent from simulation to autopilot. The RAW values of the RC channels received. The standard PPM modulation is as follows: 1000 microseconds: 0%, 2000 microseconds: 100%. Individual receivers/transmitters might violate this specification.
  Pulse Position Modulation；PPM
  如果调制信号只使载波脉冲系列中每一个脉冲产生的时间发生改变，而不改变其形状和幅度，且每一个脉冲产生时间的变化量比例于调制信号电压的幅度，与调制信号的频率无关，这种调制称为脉冲位置调制，简称脉位调制。脉冲调制是一种不连续的周期性脉冲载波的振幅、频率、强度等受到调制信号的控制而发生变化以达到传递信息信号的目的。
